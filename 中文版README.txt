🚀 免费版使用教程（100% 可运行）
1. 新建 Coze 工作流
打开 Coze → 工作台 → 工作流 → 新建空白工作流
2. 粘贴工作流配置
在画布空白处 Ctrl+A 全选
Delete 删除
把本仓库里的 workflow.json 全部内容复制
回到画布 → Ctrl+V 粘贴
等待加载完成 → 工作流自动恢复
3. 直接运行
点击右上角 运行
输入：文章标题
例如：
plaintext
男女沟通差异的真相
4. 输出结果
工作流会自动返回：
剪映草稿链接
打开即可直接编辑、导出视频。
